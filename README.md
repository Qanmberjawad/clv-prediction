# 📊 Customer Lifetime Value (CLV) Prediction

## 📌 Problem Statement
Predict the future value of customers based on historical transaction data.

## 🎯 Objectives
- Estimate Customer Lifetime Value (CLV)
- Segment customers into value tiers (Low, Medium, High)
- Provide insights for business decision-making

---

## 🧠 Project Workflow
1. Data Collection & Understanding
2. Data Cleaning
3. Feature Engineering (RFM)
4. Model Training (Multiple ML Models)
5. Model Evaluation
6. Customer Segmentation
7. Deployment (Streamlit App)

---

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- XGBoost (optional)
- Streamlit
